Optimize your images as best as possible

Stop torturing your visitors with long loading times!

Optimus.io is the best

This plugin optimizes your JPGs from the shop using optimus.io.
All new images are automatically optimized.
From our own experience, optimus.io is the best image optimizer for JPGs - even for our friend Google.

No PNGs?

We deliberately avoided implementing PNGs because optimus.io can not compress them optimally.
Although Google Insights is not interested in PNG file sizes, we still want the best possible speed for the visitor.
For the optimization of PNGs, we clearly recommend TinyPNG.

"I already have images in shop!"

No problem! There are several ways to optimize your existing images:

  1. Generate via Command alle Thumbnails neu: php bin/console sw:thumbnail:generate -f
  2. Via own Command, which searches and optimizes all images: php bin/console optimus:batch
  3. Via Backend in the media manager using "Generate Thumbnauls" in albums settings. The batch size should be "1"!

Restrictions and other recommendations

Limit 3 images/s:

Optimus.io has a limit of 3 frames per second, which slows down the upload considerably. But the result is still good.

Additional thumbnail sizes when using the responsive theme:

Article: 50x50, 86x86, 170x170, 430x430
Shopping worlds: 293x293, 931x931, 1160x1160
